## Enunciado:

### Criação e listagem de eventos futuros e passados:

Crie uma classe Evento com os atributos:
String titulo
LocalDateTime dataHora

No método main:
Crie uma List < Evento >

Adicione pelo menos 5 eventos com datas e horários diferentes
Considere que a data/hora atual é obtida com LocalDateTime.now()
Implemente as seguintes funcionalidades:

- Listar todos os eventos

- Listar apenas os eventos futuros

- Listar apenas os eventos passados
